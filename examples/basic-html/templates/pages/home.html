Welcome to Thunder Framework

A minimalist web framework made in Go from scratch.

Features:

  • Router with parameters (Go 1.22 native)
  • Middleware chain (Logger + Recover)
  • Template engine with layouts and cache
  • Static files
  • Graceful shutdown
