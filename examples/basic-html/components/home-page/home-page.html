Welcome to Thunder Framework

A minimalist web framework with component architecture, made in Go.

Features:

  • Router with parameters (Go 1.22 native)
  • Middleware chain (Logger + Recover)
  • Template engine with layouts and cache
  • Static files
  • Graceful shutdown
  • Co-located components (template + handler together)
  • Global state store (like Angular signals)
