Bienvenido a Thunder Framework

Un framework web minimalista con arquitectura de componentes, hecho en Go.

¿Qué tiene?

  • Router con parámetros (Go 1.22 nativo)
  • Middleware chain (Logger + Recover)
  • Motor de templates con layouts y caché
  • Archivos estáticos
  • Graceful shutdown
  • Componentes co-locados (template + handler juntos)
  • State store global (como signals de Angular)
