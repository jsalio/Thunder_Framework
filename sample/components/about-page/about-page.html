Acerca de Thunder Framework

Thunder es un framework minimalista para construir aplicaciones web en Go con arquitectura de componentes.

Filosofía

  • Componentes auto-contenidos: template + handler co-locados
  • Estado centralizado accesible desde cualquier componente
  • Sin magia, sin generación de código: solo Go estándar
